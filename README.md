# DOSSIER - Personal Portal

Профессиональный сайт для публикации личной информации в стиле Sherlock с темным дизайном.

## 🎨 Особенности

- **Темный дизайн** — матовые и размытые цвета
- **Профессиональное оформление** — красиво отформатированная информация
- **Admin-панель** — добавление и управление содержимым
- **Mobile-friendly** — адаптивный дизайн
- **GitHub Pages** — готово к развертыванию

## 📁 Структура

```
index.html       — Основной сайт с вашей информацией
admin.html       — Панель управления
README.md        — Этот файл
dox.txt          — Исходные данные
```

## 🚀 Развертывание на GitHub Pages

1. **Убедись что это репозиторий GitHub**
   ```bash
   git remote -v
   ```

2. **Запусти GitHub Pages**
   - Иди на Settings → Pages
   - В разделе "Build and deployment" выбери:
     - Source: Deploy from a branch
     - Branch: main, folder: / (root)
   - Сохрани

3. **Готово!**
   Сайт доступен по адресу: `https://[username].github.io/doxteample/`

## 🔐 Admin-панель

Открой `/admin.html` и введи пароль.

**Стандартный пароль:** `admin123`

⚠️ **Важно:** Измени пароль в admin.html (строка ~115)!

Найди это:
```javascript
const ADMIN_PASSWORD = "admin123";
```

И замени на свой пароль.

## 📝 Как работает

- Главная страница (`index.html`) показывает вашу информацию
- Admin-панель (`admin.html`) позволяет добавлять новые дозье
- Данные хранятся в localStorage браузера
- Каждое дозье имеет название, заголовок и содержимое

## 🎯 Как добавить новое дозье

1. Перейди на `/admin.html`
2. Введи пароль
3. Заполни форму:
   - **Название файла** — уникальное имя (англ., без пробелов)
   - **Заголовок** — отображаемое название
   - **Содержимое** — текст файла
4. Нажми "Опубликовать"
5. Обнови главную страницу

## 🛠️ Кастомизация

### Смена пароля

В `admin.html` (строка ~115):
```javascript
const ADMIN_PASSWORD = "твой_пароль_здесь";
```

### Смена цветов

В `index.html` и `admin.html` секция `<style>`:
- `#b8b8ff` — основной фиолетовый
- `#9090ff` — фиолетовый акцент
- `#7080a0` — серо-голубой

Найди и измени на свои цвета.

### Добавление секций

Скопируй блок `.section` в `index.html` и отредактируй содержимое.
