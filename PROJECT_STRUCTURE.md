## 🎯 Назначение каждого файла

### 🏠 index.html (ГЛАВНЫЙ САЙТ)
- **Что:** Основная страница с вашей информацией
- **Как использовать:** Открой в браузере или вызови из админ-панели
- **Содержит:**
  - Основные данные (ФИО, дата рождения, регион)
  - Контакты (телефон, email, ссылки)
  - Социальные сети (VK, Telegram, WhatsApp)
  - Интересы и хобби
  - Статистика Telegram
  - Подарки и регистрации

**Цвета (темная цветовая схема):**
- Фиолетовый акцент: `#9090ff`
- Серо-голубой: `#7080a0`
- Темный фон: `#0a0e27`

### 🔒 admin.html (АДМИН-ПАНЕЛЬ)
- **Что:** Приватная панель для управления контентом
- **Как использовать:** 
  1. Открой `/admin.html`
  2. Введи пароль: `admin123`
  3. Добавляй новые дозье
- **Функции:**
  - Добавление новых файлов/дозье
  - Просмотр опубликованных файлов
  - Редактирование файлов
  - Удаление файлов

**Безопасность:** Используется localStorage (только клиентская сторона)

### 📖 README.md
- **Что:** Документация проекта
- **Для кого:** GitHub, разработчики
- **Содержит:**
  - Описание проекта
  - Особенности
  - Инструкции по развертыванию
  - Как использовать admin-панель
  - Кастомизация

### ⚡ QUICKSTART.md
- **Что:** Быстрый старт за 2 минуты
- **Для кого:** Новички
- **Содержит:**
  - 3 простых шага на воздух
  - Включение GitHub Pages
  - Чек-лист

### 🎨 PERSONALIZATION.md
- **Что:** Полное руководство по кастомизации
- **Для кого:** Те кто хочет менять дизайн
- **Содержит:**
  - Смена пароля
  - Смена цветов
  - Добавление новых секций
  - Изменение информации
  - Добавление ссылок и иконок

### 📝 dox.txt
- **Что:** Оригинальный файл с вашей информацией
- **Использование:** Источник данных для index.html
- **Примечание:** Используется только как справка

### .gitignore
- **Что:** Конфигурация Git
- **Для чего:** Какие файлы не загружать на GitHub
- **Исключает:**
  - Резервные копии (*.backup, *.bak)
  - Системные файлы (.DS_Store)
  - Логи
  - node_modules (если будешь использовать)

## 🔄 Как друг с другом связаны?

```
GitHub Pages
     ↓
  index.html  ← Показывают
  admin.html  ← основную информацию
     ↓
   index.html (данные из dox.txt)
   admin.html (управляет контентом в localStorage)
```

### Поток данных:

1. **Просмотр:**
   - Юзер открывает `index.html`
   - Видит информацию (статичную или из localStorage)

2. **Добавление контента:**
   - Ты заходишь в `/admin.html`
   - Вводишь пароль
   - Добавляешь новое дозье
   - Оно сохраняется в localStorage
   - Отображается на главной странице
