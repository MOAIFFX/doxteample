## 🔑 ФУНКЦИИ ADMIN-ПАНЕЛИ

Перейди на `/admin.html` и введи пароль:

### Что можно делать:

1. **Добавить новое дозье**
   - Название файла (англ., без пробелов)
   - Заголовок (отображаемое имя)
   - Содержимое (текст файла)
   - Нажать "Опубликовать"

2. **Просмотреть опубликованные дозье**
   - Список всех добавленных файлов
   - Дата добавления

3. **Редактировать файл**
   - Нажми "Редактировать"
   - Файл загрузится в форму
   - Измени и сохрани

4. **Удалить файл**
   - Нажми "Удалить"
   - Подтверди действие

### Где хранятся данные?

Все разные дозье (после добавления через админ-панель) хранятся в **localStorage браузера**. Это означает:
- Данные сохраняются локально
- Они не видны другим
- Если очистить localStorage → все удалится

## 💻 НАСТРОЙКИ И КАСТОМИЗАЦИЯ

### Смена основной информации

Открой `index.html` → найди нужную секцию → меняй текст в `<span class="value">...</span>`

**Пример:**
```html
<span class="value">Курносова Ольга Николаевна</span>
<!-- Меняй текст между тегами -->
```

### Добавление новых секций

Скопируй этот блок и вставь перед `</div>` в конце:

```html
<section class="section">
    <h2 class="section-title">МОЯ НОВАЯ СЕКЦИЯ</h2>
    <div class="info-grid">
        <div class="info-item">
            <span class="label">Параметр</span>
            <span class="value">Значение</span>
        </div>
    </div>
</section>
```

### Добавление ссылок

В конце блока "КОНТАКТЫ" найди `<div class="links-container">` и добавь:

```html
<a href="https://твоя-ссылка.com" class="link-btn" target="_blank">
    🌐 Твоя кнопка
</a>
```

## 🐛 ЕСЛИ ЧТО-ТО НЕ РАБОТАЕТ

### Сайт не загружается:
```
1. Открой: https://[username].github.io/doxteample/
2. Подожди 1-2 минуты (GitHub Pages может медленно обновляться)
3. Нажми Ctrl+Shift+Del (очистить кэш браузера)
4. Обнови страницу (F5)
```

### Admin-панель не работает:
```
F12 → Console → Ищи красные ошибки
Если есть ошибки: проверь пароль в admin.html (строка ~115)
```

### Пароль не работает:
```
Проверь что в коде написано:
const ADMIN_PASSWORD = "admin123";
↓
И вводишь ровно: "admin123"
(Включая регистр букв!)
```

### Данные не сохраняются:
```
F12 → Application → localStorage
Если видишь данные там - OK
Если нет - localStorage заблокирована
(Попробуй инкогнито-режим)
```
